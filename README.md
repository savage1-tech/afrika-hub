# AFRIKA HUB

**Africa's Trusted Escrow E-commerce Platform**

Secure buying and selling with funds held safely until delivery is confirmed.

## Features
- Modern dark UI inspired by Bybit
- AI-powered assistant (AfriAI)
- Escrow payment protection
- Mobile-first bottom navigation
- Pan-African marketplace focus

## Security Highlights
- All transactions simulated securely
- Input validation & sanitization
- No real payment data stored
- Clear escrow flow for buyer & seller protection

## How to Run
1. Open `index.html` in any browser
2. Or visit GitHub Pages once enabled
